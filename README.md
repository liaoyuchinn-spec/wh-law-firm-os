# ⚖️ WH Law Firm OS｜律師工作系統

律師事務所實用工具集，基於 Notion 資料庫架構建立的純前端工作管理系統。

## 功能

| 模組 | 說明 |
|------|------|
| 🏠 今日工作台 | 每日三件事、P0/P1 緊急任務、即時統計 |
| 📂 案件管理 | 列表 + 看板雙檢視，依狀態篩選 |
| ✅ 任務管理 | P0–P3 優先級、關聯案件、到期日 |
| 🗓️ 行事曆 | 開庭/截止日/會議/客戶聯繫 |
| 👥 客戶管理 | 基本資料、關聯案件 |
| 🔍 司法院查詢 | 裁判書系統快速連結、法律資源 |
| 🧠 法律知識庫 | 依法領域整理的重點筆記 |

## 使用方式

直接用瀏覽器開啟 `wh-law-firm-os.html`，無需安裝套件。  
資料儲存於瀏覽器 localStorage（本機）。
